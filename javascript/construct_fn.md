# 构造函数

https://juejin.im/entry/584a1c98ac502e006c5d63b8


构造函数，就是一个普通函数，但内部使用了this变量。对构造函数使用new运算符，就能生成实例，并且this变量会绑定在实例对象上。

Js中，没有类，而是基于构造函数（constructor）和原形链（prototype）。

构造函数的三大特点：
1. 构造函数的函数名的第一个字母通常大写
2. 函数体内使用this关键字，代表所要生成的对象实例。
3. 生成对象的时候，必须使用new命令来调用构造函数。

New命令的基本原理：
new命令的作用，就是执行一个构造函数，并且返回一个对象实例。使用new命令时，它后面的函数调用就不是正常的调用，而是依次执行下面的步骤：
1. 创建一个空对象，作为将要返回的对象实例。
2. 将空对象的原型指向了构造函数的prototype属性。
3. 将空对象赋值给构造函数内部的this关键字。
4. 开始执行构造函数内部的代码。
