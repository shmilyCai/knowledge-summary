#### 布局



###### 弹性布局 - Flex（容器）    ---深入了解下

可用于设置盒子模型垂直居中

```
 display: flex;
```

设置flex后，子元素的float、clear和vertical-align属性将失效

```
align-items:center; //设置为居中
```
